# instruments

## Quick start

### 1. Clone the repo

### 2. Navigate to repo's folder

### 3. Install all the dependencies

Run `npm install` or just `npm i`

### 4. Make sure you have a running db

### 5. Change the ./config/default.json file according your settup

### 6. Run the server

`npm start`

### 7. Run the cron job

`npm run start-cron`
